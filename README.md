# Прог. теория — Android

Тренажёр по теории программирования (1 курс). Экзаменационные вопросы из конспекта
с полной теорией + дополнительный банк на 123 вопроса. Работает офлайн, интернет не нужен.

Приложение — это WebView-обёртка: вся логика лежит в `app/src/main/assets/index.html`.

## Как получить APK

### Вариант 1. Без Android Studio — через GitHub Actions

1. Залей папку проекта в репозиторий на GitHub.
2. Вкладка **Actions** → workflow **Build APK** → **Run workflow**
   (либо он запустится сам при первом push).
3. Через ~3 минуты в результатах сборки появится артефакт **progtrainer-apk** — скачай его.
4. Внутри лежит `app-debug.apk`. Перекинь на телефон и установи
   (потребуется разрешить «Установка из неизвестных источников»).

### Вариант 2. Android Studio

1. **File → Open** → выбрать эту папку.
2. Дождаться Gradle sync (скачает зависимости сам).
3. **Build → Build Bundle(s)/APK(s) → Build APK(s)**.
4. APK появится в `app/build/outputs/apk/debug/app-debug.apk`.

Либо подключить телефон по USB (с включённой отладкой) и нажать ▶ Run — приложение
установится сразу.

### Вариант 3. Командная строка

```bash
# нужен установленный Android SDK и переменная ANDROID_HOME
./gradlew assembleDebug        # или: gradle assembleDebug
```

## Обновить вопросы

Весь контент — в одном файле `app/src/main/assets/index.html`.
Банк вопросов лежит в начале блока `<script>`: массив `EXAM` (экзаменационные вопросы
с полной теорией) и `QUESTIONS` (короткий тренажёр). Замени файл — пересобери APK.

## Технические детали

- `minSdk 24` (Android 7.0+), `targetSdk 34`
- `domStorageEnabled = true` — поэтому статистика (localStorage) сохраняется между запусками
- `WebChromeClient` — нужен, чтобы работал `confirm()` при выходе из теста
- Аппаратная кнопка «Назад» уводит на главный экран приложения, а не закрывает его сразу
- Разрешение INTERNET не запрашивается — приложение полностью автономно
